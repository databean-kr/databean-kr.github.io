SMS & NMS Solutions

Integrated performance monitoring and failure troubleshooting for various heterogeneous servers & assurance of stability by managing structure, performance, operation and troubleshooting of large-scale networks

DataBean integrates all equipment at hand under the standard operation policies, and provides accurate diagnoses and high-quality service by monitoring performance and troubleshooting the failures of the assets using the ITSM technique under the ITIL Standard and analyzing the interconnected relationship.

From Databean

Since the performance of the IT equipment is comprehensively monitored, any overloaded systems can be detected and the troubleshooting can be administered quickly, thus strengthening the system operation efficiency.
Since the performance records related to CPU, memory, disk-use rate, process, network traffic, disk I/O, etc., are monitored real-time and saved in the history database, users can efficiently plan their capacity based on the accumulated data. Since the server performance data is analyzed and stored in the history database, it can be used to forecast the system resource-use volume and set the time for upgrade; and the pattern flow analysis data are provided, to check the performance asset use rate fluctuation, which is analyzed according to the flow of time. The availability rate of the service provided and the system resource-use history can be managed using various functions that manage the processes, which are the true natures of the systems, and the users can identify what service module caused the server failure. The users can analyze the web service availability and performance more practically, as the transaction performance and the conditions of failure are verified after checking the multiple web contents services where various applications are operated from the perspective of the end user.

With DataBean’s solutions, various models of network equipment from different manufacturers can be comprehensively managed, and the stability of the network can be guaranteed, since they can manage the overload / abnormal traffic, audit the operation policies of the large scale network and comprehensively inspect the large-scale networks.
